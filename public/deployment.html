Deploy on Railway (Recommended)

The easiest way to deploy the MCP server is using Railway's one-click deployment:

Detailed Railway Deployment Guide

  1. Create a Railway account
    • Go to railway.app and sign up using GitHub, Google, or Discord
    • Verify your email if required
  2. Deploy the MCP server
    • Click the "Deploy on Railway" button above
    • You'll be redirected to Railway's template deployment page
    • Click "Deploy Now" to start the deployment process
    • Railway will automatically fork the repository to your GitHub account (if you signed up with GitHub)
  3. Wait for the deployment to complete
    • Railway will automatically build and deploy the application
    • This process typically takes 2-3 minutes
    • You'll see a green "Deployed" status when it's complete
  4. Access your deployment
    • Once deployed, click on the "Deployments" tab in your project
    • You'll see a unique URL for your deployment (e.g., https://payloadcmsmcp-production.up.railway.app)
    • Click on this URL to verify your server is running

NPM Package

The MCP server is available as an npm package, making it easy to integrate into your existing projects or workflows.

Current stable version: 1.0.2

You can also use it as a dependency in your own projects:

npm install --save payload-cms-mcp
